A simple web application that allows users to search for current weather conditions in a specific location.

Description
This web application utilizes HTML, CSS, and JavaScript to provide a user-friendly interface for searching weather conditions. It listens for a form submission, fetches data from a weather API, and then displays the current weather conditions for the searched location.

Skills Displayed
Front-end development using HTML, CSS, and JavaScript
DOM manipulation and event handling
API integration and data fetching
Basic error handling and debugging
Modular code organization using multiple JavaScript files (index.js, ui.js, api.js)
CSS styling for a user-friendly interface
Note: This is a basic web application that demonstrates fundamental front-end development skills.
